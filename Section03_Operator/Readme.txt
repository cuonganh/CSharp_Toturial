Toán tử toán học:
    ++a tăng a 1, return a tăng.
    a++ tính toán, return a tăng 1.
    --a giảm a 1, return a giảm.
    a-- tính toán, return a giảm 1.

Toán tử so sánh:
    a == b  ||  a != b
    a <= b  ||  a >= b
    a >  b  ||  a <  b

Toán tử gán:
    a += c; <=> a = a + c;
    a -= b; <=> a = a - b;
    a *= b; <=> a = a * b;
    a /= b; <=> a = a / b;
    a %= b; <=> a = a % b;

Toán tử logic:
    A || B  =>  A == false or B = false => false.
            =>  else:   true.
    A && B  =>  A == true and B = true  => true.
            =>  else:   false.
    !A      =>  A == true   =>  !A = false.
            =>  A == false  =>  !A = true.
